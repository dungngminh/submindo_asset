# Submindo Plan

## Description

- Submindo aka SUBscription ReMINder(o) is a subscription management platform.
- It helps users to manage their subscriptions and track their spending.
- It also helps users to find the best deals on subscriptions.

## Features MVP

- Subscription Management: Create, Update, Delete Record
- Setup notification reminder
- View All Subscriptions
- View Upcoming Subscriptions
- View total spending by day, week, month, year
- Subscription Types Management
- Use AI to suggest better subscription plans, insight analysis.

## Tech Stack

- Mobile app: Compose Multiplatform
- Backend: No information, currently fully local
- Database: Room
- AI: Gemini API
- IAP: RevenueCat
- Notification: Local Notification
- Analytics: Firebase
